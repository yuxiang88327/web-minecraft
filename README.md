# Web Minecraft

基于网页的体素沙盒游戏，使用 Vanilla JavaScript 和 Three.js 构建。

## 功能特性

- 🏗️ 方块建造与破坏
- 🌲 程序化生成地形（山丘、树木）
- 🎮 第一人称控制器（移动、跳跃、冲刺）
- 🛠️ 工作台合成系统
- ⚔️ 装备系统（武器、盔甲、速度靴子）
- ☁️ 动态天空着色器
- 👥 多人在线模式（输入名字即可加入同一张地图）

## 运行

使用内置多人服务器运行：

```bash
node server.js
# 或
npm start
```

然后在浏览器打开 `http://localhost:8080`。

> 多人模式下，所有玩家共享同一张地图；无需注册登录，只需输入名字即可加入。

## 操作说明

| 按键 | 功能 |
|------|------|
| W A S D | 移动 |
| 空格 | 跳跃 |
| Shift | 冲刺 |
| 鼠标 | 视角 |
| 左键 | 破坏方块 |
| 右键 | 放置方块 / 打开工作台 |
| 1-9 / 0 / - | 选择方块 |
| 滚轮 | 切换方块 |
| E | 装备面板 |

## 装备配方

| 装备 | 配方 | 属性 |
|------|------|------|
| 木剑 🗡️ | 原木 + 木板 竖排 | 攻击+3 |
| 石剑 ⚔️ | 石头 + 圆石 + 木板 竖排 | 攻击+5 |
| 树叶头盔 🪖 | 3个树叶 横排 | 防御+2 |
| 树叶胸甲 🛡️ | T形树叶 | 防御+3 |
| 树叶靴子 👢 | 树叶 + 空 + 树叶 一行 | 防御+1, 速度+10% |

## 技术栈

- Vanilla JavaScript ES Modules
- Three.js v0.160.0 (CDN)
- WebGL
